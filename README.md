# Videojuegos_TestingF3_TDD

## Instalación

- Se necesita tener el lenguaje de Python instalado.
- Se necesita tener la librería de pygame, por lo general viene cono Python, pero por si acaso su instalación es en la terminal con el comando:

```bash
pip install pygame
```

- Se necesita tener las librerias sys y os instaladas, vienen por defecto en las librerías estándar por defecto de python.
- Se debe descargar en formato zip desde el repositorio de github.
- Luego se debe importar el proyecto en una carpeta.

---

## Ejecución

- Luego dentro de la carpeta se debe ir al directorio `Videojuegos_TestingF3_TDD`.
- Dentro de ese directorio se debe buscar el archivo `main.2.py` y se debe ejecutar.
- Se puede ejecutar desde un IDE como VSCode, presionando el botón de ejecutar que se encuentra en la parte superior derecha, conocido como **"Run Python File"**.
- También se puede ejecutar desde la terminal, ubicándose en el directorio anteriomente mencionado, ejecutando alguno de los siguientes comandos:

```bash
python3 main2.py
```

o

```bash
python main2.py
```

- Se abrirá una ventana donde el juego comenzará inmediatamente, generando una pieza que va cayendo poco a poco.

## Controles

- Para rotar la pieza presione la flecha X.
- Para hacer caer la pieza más rápido presione la flecha X.
- Para mover la pieza lateralmente presione la flecha X para la derecha o flecha X para la izquierda.
- Para reiniciar el juego presione la tecla ESC.

---

## Fin del juego

- El juego solo terminará cuando la siguiente pieza generada colisione con otra y saldrá un mensaje de Game Over, por lo que debe presionar ESC para poder reiniciar el juego. Para salir completamente, debe presionar la X del botón para cerrar ventana en la parte superior derecha de la ventana del juego.
